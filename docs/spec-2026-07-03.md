# 数智教材平台增强设计 - 概念图谱 / 多角色 / 仿真

> 日期: 2026-07-03
> 范围: 端到端演示闭环，纯前端 + Python 离线预处理

## 目标
从教材抽取关键词构建概念图谱（含上下级、共现、视频关联），新增进度溯源、学生/教师/家长三角色界面与仿真数据。

## 模块

### A. 概念图谱（核心）
- `scripts/extract_concepts.py`: PDF→文本→jieba 分词+停用词→TF-IDF 每章 Top-N
- 上下级: 章节标题中的概念=主题层(上); 正文中的=支撑层(下)
- 共现: 同章同时出现连边, 权重=共现章节数
- 输出 `data/concepts.json`
- `concept-graph.html`: D3 力导向图, 节点大小=度中心性, 颜色=层级, 点击侧栏显示定义+关联章节+视频

### B. 视频关联
- `data/videos.json`: 预置 ~30 常见 ML 概念→B站讲解视频链接
- 图谱节点与概念详情页跳转 B站

### C. 进度溯源
- 学生端"进度计算明细"面板
- 公式: 页数(42%)×0.4 + 章节完成(7/16)×0.3 + 习题正确率(78%)×0.2 + 时长×0.1 = 42%
- 每项可展开看来源

### D. 三角色
- 顶部角色切换器: 学生/教师/家长
- 学生 `index.html`: 自己的学习+图谱+作业+进度溯源
- 教师 `teacher.html`: 班级总览、学生分布、薄弱知识点热力图、作业提交率
- 家长 `parent.html`: 成长趋势、知识掌握雷达、学习习惯、与班级对比

### E. 仿真落点
1. 教师端: 班级在线人数跳动、今日时长滚动累加
2. 教师端: 16章×35学生掌握度热力图(色块渐变)
3. 家长端: 6维能力雷达动画填充
4. 概念图谱: 节点物理弹射入场, 悬停高亮邻居

## 不做
- 后端/真实鉴权/B站爬取/习题作答系统
- 三角色不共享实时状态(各自独立模拟数据)
